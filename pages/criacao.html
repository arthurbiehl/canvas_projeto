Aplicar animações

  • Clique para selecionar a página ou o elemento que você quer animar.
  • Na barra de ferramentas acima do editor, clique em Animar. As opções de animação serão exibidas no painel lateral.
  • Para aplicar uma animação à página selecionada, acesse a aba Animações de página .
  • Para aplicar uma animação a um elemento selecionado, escolha uma opção nas abas Animações de elementos, Animações de fotos ou Animações de texto . As abas disponíveis dependerão do elemento selecionado.
  • Para poder controlar a animação durante a apresentação, ative a opção Aparecer ao clicar.
  • Escolha entre diferentes estilos de animações e efeitos de movimento.
  • Para pré-visualizar uma animação, passe o cursor do mouse sobre ela.
  • Para aplicar uma animação, clique nela.
